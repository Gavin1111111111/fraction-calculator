# Fraction Calculator (Python + Tkinter)

A simple fraction calculator with a clean Tkinter GUI. Enter two fractions and click an operator button to add, subtract, multiply, or divide. Results are automatically simplified.

## Features
- GUI input (no terminal interaction after launch)
- Supports: **+ , - , × , ÷**
- Auto-simplifies results
- Displays whole numbers when the result is an integer (denominator = 1)

## Project Structure (recommended)
Frac-Calculator/
├── gui.py # Tkinter user interface
└── fractions_logic.py # Fraction math functions (add/sub/mult/div + simplify)
